util.domain
===========

**Entities Layer** - :ref:`clean_architecture:entities`

The util domain module contains shared business entities and enterprise-wide business rules that can be used across multiple applications in the system.

This module implements the **Entities** layer of Clean Architecture, containing:

* Enterprise-wide business rules that are least likely to change
* Core business data structures and validation logic
* Domain objects that could be used by multiple applications
