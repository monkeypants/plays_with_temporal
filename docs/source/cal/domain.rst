cal.domain
==========

**Entities Layer** - :ref:`clean_architecture:entities`

The cal domain module contains calendar-specific business entities and rules for time management, scheduling, and calendar integration.

This module implements the **Entities** layer of Clean Architecture, containing:

* Calendar entities and time block representations
* Schedule validation and business rules
* Enterprise-wide calendar concepts independent of specific implementations
